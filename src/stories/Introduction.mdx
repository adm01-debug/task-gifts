# Task Gifts Design System

Bem-vindo ao **Design System do Task Gifts** - uma plataforma de gamificação corporativa com foco em engajamento, recompensas e desenvolvimento profissional.

## 🎨 Filosofia de Design

O design system segue uma abordagem **Gaming/Corporate Gamification** com mood **energético, profissional e recompensador**.

### Cores Principais

| Token | Uso |
|-------|-----|
| `--primary` | Ações principais, CTAs |
| `--secondary` | Ações secundárias |
| `--success` | Feedback positivo, conquistas |
| `--warning` | Alertas, streaks |
| `--destructive` | Ações destrutivas |
| `--accent` | Destaques especiais |

### Cores de Gamificação

| Token | Uso |
|-------|-----|
| `--xp` | Pontos de experiência |
| `--coins` | Moedas virtuais |
| `--streak` | Sequências de atividade |
| `--rank-*` | Ranks (bronze, prata, ouro, etc.) |

## 🎯 Componentes

### Estrutura

- **UI Primitivos**: Button, Card, Badge, Avatar
- **Gamificação**: XpBar, CoinDisplay, StreakCounter, RankBadge
- **Layout**: Sidebar, Navigation, MobileBottomNav
- **Feedback**: Toast, Notifications, Achievements

### Padrões

1. **Feedback Visual**: Animações para recompensas e conquistas
2. **Responsividade**: Mobile-first com gestos nativos
3. **Acessibilidade**: WCAG 2.1 AA compliant
4. **Temas**: Suporte a dark/light mode

## 📱 Mobile

Componentes otimizados para mobile com:
- Pull-to-refresh
- Swipe gestures
- Haptic feedback
- Bottom navigation

## ♿ Acessibilidade

- Skip links para navegação por teclado
- ARIA labels em todos os componentes interativos
- Suporte a reduced motion
- Alto contraste configurável
